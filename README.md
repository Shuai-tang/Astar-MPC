# Astar-MPC：基于 A* 与模型预测控制的路径规划框架

---

## 摘要

本项目实现了一种分层路径规划框架：采用 **A\*** 算法进行全局路径搜索，采用 **模型预测控制（MPC）** 进行局部轨迹优化。全局规划得到粗粒度参考路径后，经轨迹平滑与速度规划转化为时空参考轨迹，再由 MPC 在满足动力学约束与障碍物避障的前提下进行滚动优化，得到可执行的局部轨迹。

---

## 1. 引言与研究背景

在栅格地图环境下，从起点到终点的可行路径通常由图搜索算法得到。然而，此类算法输出的离散格点序列既不满足车辆动力学约束，也缺乏时间与速度信息，难以直接用于跟踪控制。因此，需要将“几何路径”转化为“时空轨迹（Trajectory）”，并在执行层通过 MPC 进行局部优化与避障。本仓库实现了上述完整流程的代码与示例。

---

## 2. 系统架构

整体架构分为**全局规划**与**局部规划**两部分。

### 2.1 全局规划模块

1. **图搜索算法**：在栅格地图上运行 A*（传统版本与改进版本），得到从起点到终点的离散路径点序列。
2. **轨迹平滑**：对离散路径进行样条插值与滤波，得到连续、可微的几何曲线，满足动力学对曲率与平滑性的要求。
3. **时空轨迹**：在几何路径基础上建立弧长–时间（s–t）映射与速度规划，得到“何时到达何处”的参考轨迹，供 MPC 使用。

### 2.2 局部规划模块

1. **MPC（模型预测控制）**：以全局参考轨迹为输入，在有限时域内求解带动力学约束与障碍物约束的最优控制问题，输出速度与角速度等控制量，实现平滑跟踪与动态避障。

---

## 3. 轨迹平滑与时空轨迹

从“几何路径”到“时空轨迹”的转换包括以下步骤与对应输出，如表 1 所示。

| 步骤       | 输出内容           | 目的                         |
| ---------- | ------------------ | ---------------------------- |
| A*         | 离散格点序列       | 确定全局通过性               |
| 平滑       | 连续曲线函数       | 符合车辆动力学约束           |
| 速度规划   | s−t 映射关系       | 确定每个点“何时”到达         |
| MPC 提取   | 长度为 N 的矩阵   | 给控制器提供滚动优化的目标   |

**表 1** 从几何路径到 MPC 参考输入的转换流程。

A* 输出的离散网格点存在折线、不平滑及缺乏速度信息等问题。本仓库以全局航路点为输入，通过样条插值与可选高斯滤波得到平滑曲线，并计算航向角与弧长参数，为后续速度规划与 MPC 参考提取提供基础。
